# Configuração para LibreChat

O LibreChat suporta conexões SSE nativas, facilitando a integração.

1. Vá em **Settings** (Configurações).
2. Navegue até **Tools / MCP** (ou Agentes).
3. Adicione um novo endpoint (Add Endpoint).
4. Preencha a URL:

   ```
   https://mcp.apibrasil.cloud/mcp
   ```

5. Salve. As ferramentas devem aparecer na lista de tools disponíveis para os agentes.
