# Configuração para JetBrains (IntelliJ, PyCharm, WebStorm)

Atualmente, o suporte nativo a MCP no JetBrains ainda está em desenvolvimento por parte da JetBrains AI.

**A solução recomendada hoje é usar a extensão "Continue":**

1. Instale o plugin **Continue** no Marketplace da JetBrains.
2. Siga o guia na pasta `../continue/README.md` deste repositório.
3. O Continue agirá como o cliente MCP dentro do seu IDE JetBrains.
