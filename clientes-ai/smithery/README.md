# Configuração via Smithery.ai

Se o servidor estiver publicado no Smithery registry:

## Instalação via CLI

Execute no terminal:

```bash
npx @smithery/cli install apibrasil-mcp
```

## Configuração Manual

Se precisar configurar manualmente na plataforma Smithery, utilize os mesmos parâmetros de conexão SSE:

- **Type:** SSE (ou Command via ponte npx)
- **URL:** `https://mcp.apibrasil.cloud/mcp`
