# Configuração para Glue AI

Glue é uma plataforma de chat para times com suporte a Agentes.

## Configuração

1. Vá em **Integrations** ou **MCP Settings** no painel do Glue.
2. Adicione um novo servidor.
3. Insira a URL SSE direta:
   ```
   https://mcp.apibrasil.cloud/mcp
   ```
4. O Glue deve reconhecer automaticamente as ferramentas e disponibilizá-las para os agentes do time.
