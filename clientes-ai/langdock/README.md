# Configuração para LangDock

LangDock é uma plataforma enterprise para LLMs.

## Configuração

1. Painel de Admin > **Integrations**.
2. Selecione **Model Context Protocol**.
3. Add Endpoint.
4. URL: `https://mcp.apibrasil.cloud/mcp`
5. Defina as permissões (quem pode usar as ferramentas da APIBrasil).
