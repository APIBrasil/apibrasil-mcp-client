# Configuração para Cherry Studio

Cherry Studio é um cliente desktop elegante para múltiplos modelos.

## Configuração

1. Clique no ícone de **Engrenagem** (Configurações).
2. Vá em **Tools & MCP**.
3. Clique em **Add Server**.
4. Tipo: **SSE**.
5. URL: `https://mcp.apibrasil.cloud/mcp`
6. Certifique-se de habilitar o switch do servidor após adicionar.
