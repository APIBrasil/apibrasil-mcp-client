# Configuração para MindPal

MindPal é um cliente web focado em fluxos visuais.

## Configuração

1. No dashboard, clique em **Connectors** ou **MCP**.
2. Escolha **Add Remote Server**.
3. URL: `https://mcp.apibrasil.cloud/mcp`
4. Autorize a conexão.
5. Agora você pode arrastar as ferramentas da APIBrasil para dentro dos seus fluxos de pensamento.
