# Configuração para TypingMind

TypingMind é uma interface web popular para ChatGPT/Claude que suporta plugins.

## Configuração via Plugin MCP

1. Vá em **Plugins**.
2. Clique em **Add New Plugin**.
3. Escolha **Import from MCP**.
4. Se o TypingMind estiver rodando localmente (versão Desktop) ou com o "TypingMind Extension" no browser, ele pode acessar a URL.
5. URL: `https://mcp.apibrasil.cloud/mcp`

*Nota: Se usar a versão web pura sem extensão, pode haver bloqueio de CORS. Recomenda-se o App Desktop do TypingMind.*
